## 2025-02-12 - [Color-coded Status Badges]
**Learning:** Mapping complex data states (like clinical trial statuses) to visual cues (Bootstrap badges) significantly improves scanability of data tables. When using Quarto f-strings to generate Python code in QMD files, literal curly braces must be doubled to avoid SyntaxErrors during the generation process.
**Action:** Always use color-coded badges for status fields in data-heavy interfaces. Be extra careful with nested f-strings and literal braces in code-generating scripts.
